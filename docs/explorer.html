How to export a filtered country list — examples & tool tips

Every tab has an Export CSV button. It downloads exactly the rows currently shown in the table — all filters, dropdowns, and search terms are applied before export. The file is UTF-8 CSV, compatible with Excel, R (readr/read.csv), Python (pandas), Stata, SPSS, and Power BI. The filename includes the classification system and row count so you always know what you downloaded.

UN M49 / Geoscheme — examples

All LDC countries with geographic codes
Set Special group → LDC only → Export → columns: country_name_en, iso3, iso2, m49, region, sub_region, intermediate_region, is_ldc, is_lldc, is_sids. Useful as a master list for any analysis that requires UN M49 codes alongside LDC/LLDC/SIDS flags.
All countries in a specific sub-region
Set UN Sub-region → Eastern Africa → Export → precise 20-country list with full geographic hierarchy. Combine with the search box to further narrow by name.

World Bank — examples

Low-income economies in Sub-Saharan Africa
Set Income level → Low income + WB Region → Sub-Saharan Africa → Export → the exact list used in World Bank operational and poverty targeting work.
All IDA-eligible economies
Set Lending type → IDA → Export → countries eligible for concessional IDA financing (wb_income_name, wb_lending_name, capital_city included per row).
Upper-middle-income economies globally
Set Income level → Upper middle income → Export → used to identify countries approaching high-income status or recently graduated from ODA eligibility.

OECD DAC — examples

All ODA-eligible LDCs in Africa
Set DAC group → LDCs + UN Region → Africa → Export → columns include dac_group, wb_income_hint, reporting_year. Use directly for ODA targeting, donor reporting, or OECD peer-review submissions.
All ODA-eligible countries in a specific reporting year
No filter needed — the reporting_year column in the export shows the year each country was added to or last confirmed on the DAC list.

World Bank FCS — examples

Conflict-affected countries only
Set Category → Conflict → Export → 21-country list with wb_region, wb_income_name, wb_fcs_category, wb_fcs_fy. Useful for humanitarian programming, risk analysis, and IDA conflict windows.
FCS countries in a specific WB region
Set WB Region → Sub-Saharan Africa (or any region) → Export → fragile and conflict-affected states within that region only. Combine with Category filter to separate Conflict from Institutional Fragility.

UN SDG Regions — examples

All countries in a specific SDG group
Select e.g. Sub-Saharan Africa in the SDG group dropdown → Export → 53-country list. The sdg_groups column lists all groups each country belongs to, separated by ; — so one row per country, all memberships visible at once.
Splitting the sdg_groups column into multiple rows
Excel: select the column → Data → Text to Columns → delimiter ;
R: tidyr::separate_rows(df, sdg_groups, sep = ";")
Python: df["sdg_groups"].str.split(";").explode().reset_index(drop=True)
Stata: split sdg_groups, parse(";")
Power BI: Split Column → By Delimiter ; → Into Rows

General tips

Match counts before exporting
The Showing box and the Export CSV note both display the exact row count. If the count looks wrong, check your active filters — each dropdown narrows the list independently.

UN M49 Standard Country or Area Codes

The foundational UN classification system published by the UN Statistics Division (UNSD). Covers all 248 UN statistical countries and areas — including territories and dependencies that may not be members of the UN or classified by the World Bank or OECD. Provides the global geoscheme (global → region → sub-region → intermediate region), plus three special designations: LDC (Least Developed Countries), LLDC (Landlocked Developing Countries), and SIDS (Small Island Developing States).

World Bank Country & Lending Groups

The World Bank classifies 215 economies by income level and lending eligibility. 33 UN statistical territories are not included — mostly French and British overseas territories, Western Sahara, and similar — so the WB count for the African continent is 54, not 60. Income levels are updated each July using Atlas GNI per capita. Lending type shows IDA/IBRD eligibility.

⚠ WB region ≠ UN region: The World Bank labels its Africa region "Sub-Saharan Africa" (48 countries). The remaining 6 North African countries (Algeria, Egypt, Libya, Morocco, Tunisia, Sudan) appear in the WB region "Middle East, North Africa, Afghanistan & Pakistan". World Bank — Fragile and Conflict-affected Situations (FCS)

The World Bank publishes an annual list of countries classified as Fragile and Conflict-affected Situations (FCS). Countries are grouped into two categories: Conflict (active conflict with significant impact on governance and service delivery) and Institutional and Social Fragility (weak institutions, social cohesion, or high vulnerability without active conflict). The list informs IDA resource allocation and World Bank operational policies.

UN SDG Geographic Regions

The UN SDG monitoring framework uses its own geographic grouping system, different from UN M49 and World Bank regions. SDG groups are overlapping — a country can belong to multiple groups simultaneously (e.g. "Africa", "Sub-Saharan Africa", "Eastern Africa", and "LDC Africa" at once). There are — distinct SDG groups.

Why SDG "Africa" = 60 but World Bank "Africa" = 54: The SDG group "Africa" covers the full continental Africa — all 60 countries per UN M49, including North African and overseas territories. The World Bank's "Sub-Saharan Africa" region covers only 48 countries (North Africa sits in WB's MENA region, and 6 territories have no WB classification at all). These are different groupings answering different questions. For Sub-Saharan Africa specifically, select "Sub-Saharan Africa" in the filter below (53 countries).
