Core Datasets

  • countries_master.csv: country-level master table
  • country_group_membership.csv: long-format memberships
  • country_classification_library.csv: full denormalized library
  • sources.csv and run_manifest.json: provenance + snapshot metadata

Automation

  • GitHub Pages auto-deploys updated docs.
  • All outputs are versioned in repository history.
